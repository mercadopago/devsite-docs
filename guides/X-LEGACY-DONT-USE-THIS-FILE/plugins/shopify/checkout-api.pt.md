# Checkout Transparente

O [Checkout Transparente](/developers/pt/guides/checkout-api/introduction) do Mercado Pago permite que todo o processo de compra aconteça no ambiente da sua loja, sem a necessidade de um redirecionamento para uma página externa para concluir o pagamento.

> WARNING
>
> Importante
>
> Caso você já tenha uma versão anterior do Checkout Transparente instalada, será necessário desinstalá-la antes de configurar a nova versão. Para isso, acesse o painel administrativo da sua Shopify e clique em **Apps > Excluir Checkout Transparente Mercado Pago**.
